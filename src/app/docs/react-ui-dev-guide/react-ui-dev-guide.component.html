This guide presents how to use ApiRTC set of React components to speed-up your React client application development.

React UI Library

Important ! If you are not aware of the available components, please check the demo app and/or the demo repository.

useApiRTC

Hook used to interface apiRTC javascript library with React components.

Parameter with * are required
Parameter Description Type Default value 
api_key*
 Your api key claimed on https://cloud.apirtc.com, can also be an empty string string 
-

LoginForm

Display a login form used to identify user with a username/password pair. This component is not intended for user's connection by apiKey !

The login prop is retrieved thanks to the hook useApiRTC with apirtc.login

Props with * are required
Props Description Type Default value 
labels
 Edit labels of fields {{"{ username?: string; usernameTooltip?: string | undefined; password?: string | undefined; passwordTooltip?: string | undefined; submit?: string | undefined; cancel?: string | undefined; \}"}} 
-
 
title
 Set title at the top of the login form ReactNode 
-
 
subTitle
 Set subtitle at the top of the login form ReactNode 
-
 
showTooltipHelpPassword
 Display or not a helping tooltip about the password field boolean 
false
 
onSubmit*
 Function executed when the login form is submitted {{"(username: string, password: string) => Promise\"}} 
-
 
onCancel
 Function executed when the user click on the cancel button () => void 
-
