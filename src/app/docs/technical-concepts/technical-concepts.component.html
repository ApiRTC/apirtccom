Technical concepts

Any ApiRTC application, involves at least two software parts, the client application and ApiRTC platform.

Client application

The client application is the web or mobile application final users are using to interact together. It allows the user to share messages or media, and it displays messages or media from the other participants

ApiRTC platform

ApiRTC cloud

ApiRTC cloud is responsible for the signaling.
