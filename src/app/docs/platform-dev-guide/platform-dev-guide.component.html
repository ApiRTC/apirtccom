Learn how to interact with ApiRTC Platform.

ApiRTC platform is compound of software layers to enables communication between people through client applications. ApiRTC REST Api enables interaction between your backend and ApiRTC backend.

The most common interaction is provisioning the services you have registered for. This includes the User Management.

The REST Api also allows to retrieve communication data, such as recorded videos

Use subaccounts to simplify configuration, reporting, and billing of different projects.

Organizing your communication traffic by use case is paramount to simplifying the billing allocation process. The Subaccounts API allows developers to spin up new subaccounts for each new use case and business unit, allowing billing and traffic to be allocated efficiently at the end of every month.
